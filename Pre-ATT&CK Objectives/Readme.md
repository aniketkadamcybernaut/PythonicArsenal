# Welcome to Pre-ATT&CK Objectives

This folder contains specific files related to "Pre-ATT&CK Objectives" outlines the foundational stages of cyber threat activity, focusing on reconnaissance, initial access, and target selection before the actual attack. This content provides insights into understanding and mitigating potential threats early in the cyber kill chain.

For detailed information and instructions, please refer to the main README file located in the root directory of the repository.And also check for Requirement file located in the root directory of the repository
